# Aircond Servis

Landing page untuk perkhidmatan servis aircond di Malaysia. Dibina menggunakan [Astro](https://astro.build/) dan Tailwind CSS.

## Mula

```bash
npm install
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview production build
```

## Struktur Projek

```
src/
  layouts/Layout.astro   # Layout utama (navbar, footer, WhatsApp button)
  pages/index.astro      # Halaman utama
public/
  favicon.svg
```
